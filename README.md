# Quantum-icon-theme
Linux icon theme

# Based On
* Papirus (some assets)
* Fluent by vinceliuice
* Moka by snwh (Design)
* breeze-extended-alt (Trash Can Design)

# Features

* Light Panel
* Pixel Perfect Icons
* Pop!_OS COSMIC Support

# Install

Open terminal, and run install.sh file.
